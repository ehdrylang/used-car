## 중고차 배점표

### 용도이력
- 렌트, 택시 등 용도이력이 존재하는 경우 -15점
- 리콜필요한 것은 점수 없음

### 사고/보험이력
- 자차보험미가입기간 존재 : -10점
- 무사고 + 보험 이력 없음 : + 20점
- 사고/전손/침수: -30점
- 단순수리 또는 무사고 + 보험 이력 있는 경우
  - 내차 보험 수리 이력 평균가격이 100만원 이하(국산차) : +10점
  - 내차 보험 수리 이력 평균가격이 200만원 이하(수입차) : +10점
- 비공개 : -20점
- 그 외 점수 없음.

### 1인소유
- 소유주 변경이력이 1건 이하는 매매상사로 이전한 것이 있을 수 있어서 1인소유차량으로 봄 : +10점
- 2건 : 0점
- 3건 이상 : -10점
- 2건 이상인 경우 중 소유주 변경 사이 갭이 1년 이내인 경우 : -5점

### 판매자 정보
- KCAR 판매자 정보가 없음 : +3점
- ENCAR만 다양한 딜러가 존재하므로 딜러의 1년간 판매 대수가 100대 이상이다 : +5점
- encar 에 등록된지 5년이상 지난 딜러인 경우 : +5점
- encar 에 등록된지 10년이상 지난 딜러인 경우 : +10점
- 판매자가 개인인 경우 : -5점

### 주행거리
- 1년에 평균 15,000km 로 책정하고 평균에 따른 점수 부과
- 연평균 10,000km 이하 : +10점
- 연평균 15,000km 이하 : +0점
- 연평균 15,000km 이상 20,000 미만 : -5점
- 연평균 20,000km 이상 : -10점

### 연식
- 3년 이하 : +5점
- 3~7년 : 0점
- 8년이상 : -5점

### 성능점검상
- 주요 골격에 문제가 있다 : -10점
- 외판 또는 본네트 교환, 좌우휀더 교환이 있다 : -5점
- 누유가 없다 : +3점

### 본문
- "완전무사고" 라는 표현이 있는 경우 : +5점
