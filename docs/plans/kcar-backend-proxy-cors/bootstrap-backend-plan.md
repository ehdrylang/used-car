# Sub Task Plan: 백엔드 부트스트랩

## 목표
- `backend` 실행 골격을 만들고 로컬에서 서버를 띄울 수 있는 최소 환경을 준비한다.

## 작업 범위
- `backend` 디렉터리 생성
- `package.json` 스크립트 구성 (`dev`, `start`, `test`)
- 서버 엔트리 파일 생성
- 환경변수 로딩 기본 구조 생성

## 구현 순서
1. 런타임/패키지 초기화
2. `/health` 헬스체크 라우트 추가
3. 포트 및 공통 설정 파일 분리
4. 로컬 실행 확인

## 완료 기준
- 서버가 지정 포트에서 정상 기동된다.
- 헬스체크가 200 응답을 반환한다.
