# Sub Task Plan: KCAR 브랜드 선택 화면 및 상태 처리

## 목표
- KCAR 화면 좌측 패널에서 브랜드를 국산/수입으로 표시하고 단일 선택 상태를 관리한다.

## 작업 범위
- 브랜드 API 연동 함수 작성
- 로딩/성공/빈 상태/오류 상태 UI 처리
- 브랜드 단일 선택 상태 관리
- 우측 안내 문구 표시

## 구현 순서
1. API 서비스 모듈에 브랜드 조회 함수 추가
2. 페이지 진입 시 브랜드 목록 fetch
3. `carType` 기준으로 국산/수입 그룹 렌더링
4. 브랜드 클릭 시 단일 선택 상태 반영
5. 오류 시 재시도 동작 연결

## 완료 기준
- 브랜드 목록이 좌측 패널에 정상 노출된다.
- 단일 선택 상태가 명확히 반영된다.
- 로딩/오류/빈 데이터 상태가 명세대로 동작한다.
- 우측 영역에 모델 선택 예정 안내가 표시된다.

## 의존성
- Task 1 라우팅 골격
- Task 3 API 계약 검증 결과

## 리스크
- 응답 데이터량이 많아질 경우 렌더링 성능 고려가 필요.
